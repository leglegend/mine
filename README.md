# business2.0

> 坚果卡包商家版2.0

## 运行指引

``` bash
# 初始化小程序
npm install

# 初始化cropper组件
npm install mpvue-cropper --save

# 加载sass
npm i sass-loader@6.x --save-dev
npm install node-sass@latest

# 在localhost:8080端口运行小程序
npm run dev

# 构建小程序
npm run build

# 小程序构建报告
npm run build --report
```


关于本小程序的详细信息参见 [坚果卡包商家版介绍](https://www.nutcards.com/business.html).
